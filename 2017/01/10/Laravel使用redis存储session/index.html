发表于 2017-01-10

默认情况下，Redis 服务会提供 16 个数据库，Laravel 使用数据库 0 作为缓存和 Session 的存储。
在使用的过程中觉得这个默认的设置挺不合理，因为当你在执行命令php artisan cache:clear 的时候，会把 Session 也连带清除了，导致所有用户需要重新登录。
