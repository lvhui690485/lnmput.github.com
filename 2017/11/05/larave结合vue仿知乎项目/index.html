larave结合vue仿知乎项目

发表于 2017-11-05

失去就好像，你将余生写一首长诗，却不能提及一个与她有关的字……

简介

基于laravel5.3 vue2.0开发的仿知乎项目，实现基本的登录注册，发布问题，关注问题，回答问题，评论问题，发送私信，系统发送邮件通知，使用千牛存储图片，使用阿里云提供的短信服务发送验证码等基本功能.
